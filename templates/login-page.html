Welcome to wordsearch app

This is a database that you can use to store words and their definitions as you learn them.

Im an amateur and I built this app to help me learn both Dutch and web-development in Golang, you can see the source code here.

All of your data is stored as plain text in my database with NO encryption. So I (and potential malicious actors) have full access to it. And I'm not knowledgeable enough to implement all of the best practices for data and account protection. Never and I mean NEVER use this app with any of your real passwords.

And don't store any data that could be harmful to you when leaked.
